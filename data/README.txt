Dataset — Panorama Stitcher (Groupe_Alaoui)
============================================
This folder contains the input images used for panorama stitching.

Structure:
  images/set1/   — first set of overlapping photos (3-4 images)
  images/set2/   — second set (optional, for additional testing)
  output/        — generated panorama results

Capture guidelines:
- Overlap between consecutive images: ~40%
- Keep the camera at the same height, rotate horizontally only
- Avoid moving subjects in the scene
- Consistent exposure between shots recommended
